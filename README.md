# Airport Safety ETL

ETL pipeline project for aviation safety events analysis.

## Project goal

Создание ETL-процесса для обработки данных о событиях, связанных с безопасностью полетов в международном аэропорту.

Основные этапы:

- загрузка исходных данных;
- проверка качества данных;
- очистка и преобразование данных;
- загрузка данных в PostgreSQL;
- подготовка данных для аналитики.

## Technologies

- Python
- Pandas
- PostgreSQL
- SQL
- Git
